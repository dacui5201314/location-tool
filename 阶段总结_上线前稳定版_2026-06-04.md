## 后端生产部署注意事项

1. **同步代码**：将本地 `backend/` 目录同步到服务器 `/www/wwwroot/location-tool/backend/`
2. **安装依赖**：`pip3 install -r backend/requirements.txt`
3. **确认微信支付配置**：
   - `wx_app_id` = `wx_mini_appid` = `wxc8f7f6a2e29e7bb4`
   - `wx_notify_url` = `https://www.oliver188.top/api/pay/wechat/notify`
   - 商户号与 AppID 必须属于同一微信支付主体
4. **重启后端**：
   ```bash
   fuser -k 8000/tcp ; sleep 1
   cd /www/wwwroot/location-tool/backend
   nohup python3 -m uvicorn main:app --host 0.0.0.0 --port 8000 > /dev/null 2>&1 &
   ```
5. **Nginx** 需代理 `/assets/` 到后端或直接 serve `backend/storage/assets/`
6. **微信公众平台**配置：
   - request 合法域名：`https://www.oliver188.top`
   - downloadFile 合法域名：`https://www.oliver188.top`
